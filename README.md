# Screen Time vs GPA Analysis

## Project Overview
This project examines the relationship between daily screen time and students’ academic performance (GPA).

## Data Collection
The dataset was collected through a survey of university students. A total of 35 responses were gathered.

## Methodology
The data was cleaned and processed using Python. Non-numeric responses such as ranges (e.g., "6-7") were converted into numerical values.

## Results
The analysis shows that there is no strong relationship between screen time and GPA.

However, study time shows a stronger positive relationship with GPA.

## Conclusion
Screen time alone is not a strong predictor of academic performance. Study habits play a more important role.

## Tools Used
- Python (pandas, matplotlib)
